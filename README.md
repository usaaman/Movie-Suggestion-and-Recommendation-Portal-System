## 📌 About The Project

> **"Never waste time deciding what to watch again!"**

The **Movie Suggestion and Recommendation Portal** is a dynamic web application built with PHP and MySQL that helps users discover movies tailored to their personal tastes. It uses intelligent recommendation algorithms to suggest films based on user ratings, viewing history, and similar users' preferences.

Whether you're a movie enthusiast looking for your next watch or someone who suffers from decision paralysis, this portal is your ultimate movie companion! 🍿

---

## ✨ Features

### 👤 User Features
| Feature | Description |
|---------|-------------|
| 🔐 **Authentication** | Secure registration and login system |
| 🎯 **Smart Recommendations** | Personalized suggestions using hybrid filtering |
| ⭐ **Rate & Review** | Rate movies (1-5 stars) and write reviews |
| 🔍 **Advanced Search** | Search by title, genre, cast, or director |
| 📋 **Watchlist** | Save movies to watch later |
| 📊 **User Dashboard** | View ratings, watchlist, and recommendations |

### 👑 Admin Features
| Feature | Description |
|---------|-------------|
| 🎬 **Movie Management** | Add, update, delete movies |
| 🏷️ **Genre Management** | Manage movie categories |
| 👥 **User Management** | View and manage user accounts |
| 📝 **Content Moderation** | Monitor and moderate user reviews |
